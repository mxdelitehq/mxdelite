# Mxdelite

Official website for Mxdelite — Elite Digital Agency.

## Structure

```
mxdelite/
├── index.html        # Main HTML
├── css/
│   └── style.css     # All styles
├── js/
│   └── main.js       # Grain effect, scroll reveals, form submission
├── .gitignore
└── README.md
```

## Deployment

Drag the entire folder into [Netlify](https://netlify.com) to deploy.

## Form

Contact form submits to Google Sheets via Google Apps Script.
Script URL is set in `js/main.js` under `SCRIPT_URL`.

## Fonts

- **Montserrat** — Logo, UI, body (Google Fonts)
- **Raleway** — Headlines (Google Fonts)
